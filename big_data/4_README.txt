Key features to consider when using a Big data technology stack

1) Integration - How easy is it to integrate with you current software stack
2) Handling Big-Data - How easy is it to access big data. Is it viewable easlily over the web, through REST-API's?, through some custom UI etc
3) Real-time - Is data available realtime? How soon is data replicated across your complete clustor?
4) Cost --> How efficient is cost? Typically it takes more resources to setup products but maintainability should be really low.
5) Scalability?
6) Realiability?
7) Any inbuilt Analytics?
8) Any data pre-processing capability
9) Any algorithmic query
